### 💻 Featured Projects

**[Research MAS: Production-Grade Multi-Agent AI System](https://github.com/Monukumar-21/Research_MAS)**
*   **What it is:** An end-to-end academic research system orchestrated by 8 specialized AI agents (Planner, Retriever, Writer, Critic, etc.) routed by a "Supervisor" node.
*   **Tech Stack:** `Python`, `FastAPI`, `Streamlit`, `LangGraph`, `MongoDB`, `Pinecone`
*   **Engineering Highlights:** Built a resilient system architecture capable of hot-swapping LLMs (Gemini, OpenAI, Groq). Solved critical production issues including graceful degradation during API rate limits (429 errors), preventing zombie processes by isolating OS signals in the FastAPI/Streamlit dual-boot, and handling multi-LLM embedding conflicts dynamically.

**[Amazon Product Recommendation System](https://github.com/Monukumar-21/amazon-product-based-recommendation-system)** | *[Live Demo](https://amazon-electronics-recommendation-system.streamlit.app/)*
*   **What it is:** A content-based, item-to-item recommendation engine built on the Amazon Electronics review dataset.
*   **Tech Stack:** `Python`, `Scikit-Learn`, `TF-IDF`, `Cosine Similarity`, `Streamlit`
*   **Engineering Highlights:** Implemented NLP preprocessing and vector similarity search. Actively researched industry-scale solutions to the cold-start problem, laying the groundwork for transition into Two-Tower/Dual-Encoder deep learning architectures. 

**[Movie Sentiment Analyzer](https://github.com/Monukumar-21/sentiment_analysis)** | *[Live Demo](https://moviesentiment-analyzer.streamlit.app/)*
*   **What it is:** An NLP pipeline comparing classical ML (Logistic Regression, Decision Trees) against Deep Learning (RNN, LSTM) for movie review classification.
*   **Tech Stack:** `PyTorch`, `Scikit-Learn`, `Hugging Face`, `Streamlit`
